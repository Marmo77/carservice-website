# Autosc Website

A React + TypeScript project built with Vite, focused on modern UI design and interactive features.

---

## About The Project

Autosc Website is a self-training React application designed to practice and showcase modern web UI design using CSS and React components. The project includes key features such as:

- Responsive and modern UI layouts
- Interactive Shop and Achievements components
- State management with React hooks and localStorage persistence
- TypeScript for type safety and better developer experience

This project helps improve skills in React, TypeScript, and UI/UX design while building a functional web app.

---

## Features

- **Modern UI Design:** Responsive layouts and polished styling using CSS and Canva-inspired design principles.
- **Shop Component:** Allows users to purchase upgrades with dynamic pricing and state persistence.
- **Achievements Component:** Tracks user progress and achievements with interactive UI.
- **State Persistence:** Uses `localStorage` to save coins, upgrades, clicks, and worker power between sessions.
- **TypeScript & React Hooks:** Strong typing and hooks like `useState` and `useEffect` for state and side effects.
- **Version Control:** Managed with Git and hosted on GitHub for collaboration and versioning.
